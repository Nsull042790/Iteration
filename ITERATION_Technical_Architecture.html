GAME.JS - MAIN LOOP
gameLoop(timestamp) { const deltaTime = timestamp - this.lastTime; this.lastTime = timestamp; if (!this.isPaused) { this.update(deltaTime); // Update all entities this.checkCollisions(); // Physics/collision detection } this.render(); // Draw everything requestAnimationFrame((t) => this.gameLoop(t)); }
FRAME RATE

Target: 60 FPS via requestAnimationFrame. DeltaTime used for frame-independent movement. Physics updates capped at consistent tick rate to prevent tunneling.

PLAYER UPDATE CYCLE
  • Read input state from InputHandler
  • Apply movement forces (velocity += input * speed)
  • Apply gravity (GAME_CONFIG.GRAVITY = 0.6)
  • Handle attack state machine (cooldowns, hitboxes)
  • Deduct cycle costs for actions
  • Update invincibility frames if damaged

AABB COLLISION

All collision detection uses Axis-Aligned Bounding Boxes. The Physics.checkCollision(a, b) function returns collision data including penetration depth for resolution.

PHYSICS.JS - COLLISION CHECK
static checkCollision(a, b) { return a.x < b.x + b.width && a.x + a.width > b.x && a.y < b.y + b.height && a.y + a.height > b.y; } static resolveCollision(entity, platform) { // Push entity out of platform // Set isGrounded = true if landing on top // Handle one-way platforms (only collide from above) }

UPGRADE TIERS

Each weapon has 5 tiers: TIER_1 → TIER_2 → TIER_3 → TIER_4 → TIER_MAX. Each tier adds ~20% damage and unlocks new visual effects. At MAX tier, a unique ability is unlocked.

AUDIO SYSTEM

All audio is synthesized in real-time using the Web Audio API. No external sound files required.

AUDIO.JS - SOUND SYNTHESIS
class AudioSystem { constructor() { this.ctx = new (AudioContext || webkitAudioContext)(); this.masterGain = this.ctx.createGain(); this.masterGain.connect(this.ctx.destination); } playSword() { // Oscillator: sawtooth wave // Frequency sweep: 800Hz → 200Hz over 100ms // Envelope: quick attack, fast decay } ensureReady() { if (this.ctx.state === 'suspended') { this.ctx.resume(); } } }
  • playSword() - Sawtooth sweep for melee attacks
  • playHit() - White noise burst for damage
  • playJump() - Quick pitch rise
  • playBossMusic() - Looping bass drone with arpeggios
  • playVictory() - Major chord fanfare

PROCEDURAL GENERATION

ROOM.JS - LEVEL GENERATION
function generateRandomRoom(level, zoneIndex) { const room = new Room(); const zoneColors = GAME_CONFIG.ZONE_COLORS[zoneIndex]; // 1. Add floor platforms addFloorPlatforms(room, zoneColors); // 2. Add vertical sections with gaps addVerticalPlatforms(room, level, zoneColors); // 3. Add floating platforms (more at higher levels) const platformCount = 5 + Math.floor(level / 2); addFloatingPlatforms(room, platformCount, zoneColors); // 4. Add decorative elements addDecorations(room, zoneColors); return room; }
PLATFORM OVERLAP PREVENTION

New platforms check against existing platforms using checkOverlap(). If overlap detected, position is adjusted or platform is skipped. Minimum gap of 40px maintained between platforms.

BUILD & DEPLOY

NO BUILD STEP REQUIRED

ITERATION is built with vanilla JavaScript. No transpilation, bundling, or npm required. Simply serve the files via any HTTP server or open index.html directly in a browser.

CACHE BUSTING

All script tags include version query parameters:

<script src="src/core/game.js?v=1.8.0"></script>

Increment version number to force browser cache refresh on updates.

MOBILE SUPPORT
  • Touch controls auto-detect mobile via user agent
  • Virtual joystick for movement
  • Touch buttons for jump, attack, interact, weapon switch
  • Responsive canvas scaling
